# Chatroom React

This project is built with React and Vite.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Starts the Vite development server. By default, Vite prints the local URL in the terminal after startup.

### `npm run build`

Builds the app for production to the `dist` folder.

The build is optimized for production and ready to deploy.

### `npm run preview`

Serves the production build locally so you can verify the generated output before deployment.
